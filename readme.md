
## Run this project

- composer install
- OPTIONAL: 'npm install' OR 'yarn'
- OPTIONAL: npm run production
- Set .env file with this variables filled
  DB_CONNECTION=
  DB_HOST=
  DB_PORT=
  DB_DATABASE=
  DB_USERNAME=
  DB_PASSWORD=
- php artisan migrate:fresh --seed
